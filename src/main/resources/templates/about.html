About

The L20 Polling System is a platform developed for the software engineering course at Università di Pavia by, in alphabetical order:

  • Francesco Filpa
  • Francesco Maffia
  • Kasian Nitse
  • Andrea Portaluppi
  • Emanuele Simonetti Rosario

This platform allows you to create a personal account, verified by e-mail, and create polls you can share with the community, or with a selected number of other registered accounts – colleagues, friends or family.
